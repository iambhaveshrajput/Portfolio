# 🚀 AI Portfolio - Bhavesh Rajput

A high-performance, full-stack portfolio featuring an **AI Assistant** built with a custom RAG (Retrieval-Augmented Generation) pipeline.

**🔗 Live Website:** [https://bhaveshrajputportfolio.netlify.app/](https://bhaveshrajputportfolio.netlify.app/)

---

## 🛠 Tech Stack
* **Frontend:** HTML & Tailwind CSS (Deployed on **Netlify**)
* **Backend:** FastAPI & Python (Deployed on **Render**)
* **AI Engine:** Google Gemini 3 Flash
* **Vector Database:** Pinecone (Serverless)
* **Orchestration:** LangChain 0.3

## ✨ Key Features
* **Smart Assistant:** A chatbot that answers questions specifically about my projects and research using a 3072-dimension vector search.
* **Project Showcase:** Deep dives into **CoffeeConnect** (Research Paper) and **NeuroTask AI**.
* **Responsive Design:** Fully optimized for mobile and desktop viewing.

## 📂 Project Structure
* `/frontend`: React source code and UI components.
* `/backend`: FastAPI routes and CORS configuration.
* `chatbot.py`: The core RAG logic and Gemini integration.
* `portfolio.md`: The knowledge base used by the AI.
